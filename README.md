
# Core Banking Ledger System (Python CLI)

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![JSON](https://img.shields.io/badge/Data-JSON-lightgrey?style=for-the-badge)
![OOP](https://img.shields.io/badge/Architecture-OOP-blue?style=for-the-badge)

A robust terminal-based Bank Management System engineered in Python utilizing structural **Object-Oriented Programming (OOP)** and persistent **JSON data storage**. This project demonstrates a production-grade approach to handling sensitive data and complex logic flow.

## 🌟 Key Features
- **Account Management:** Secure creation and management of user accounts.
- **Transaction Processing:** Real-time deposit, withdrawal, and transfer logic with balance validation.
- **Persistent Storage:** All data is saved in structured JSON format, ensuring data persistence across sessions.
- **Robust Security:** Built-in error handling and input validation to prevent invalid operations.
- **Clean Architecture:** Divided into modular classes for scalability and maintenance.

## 🛠 Tech Stack
- **Language:** Python 3.x
- **Storage:** JSON (Structured File I/O)
- **Paradigm:** Object-Oriented Programming (OOP)

## 🚀 Getting Started

### Prerequisites
- Python 3.11+

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/hafizabdulaziz/bank-management-system.git
   ```
2. Navigate to the directory:
   ```bash
   cd bank-management-system
   ```
3. Run the application:
   ```bash
   python main.py
   ```
